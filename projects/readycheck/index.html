ReadyCheck

A CLI for repeatable, audit-friendly connectivity validation that compares observed network behavior to declared intent.

View repo

Summary

ReadyCheck focuses on evidence collection for connectivity validation. It separates raw check success from policy outcome and emits artifacts intended for operations handoff and change review.

Current framing

  • YAML-driven source, destination, and check matrix
  • Supports tcp, tls, http, ssh, and dns checks
  • Produces JSON, CSV, HTML, summary, and baseline diff artifacts
