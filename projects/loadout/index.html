Loadout

A Windows infrastructure workstation setup tool built as a portable, self-updating PowerShell and WPF application.

View repo

Summary

Loadout is positioned around a practical workstation problem: selecting, previewing, installing, updating, and uninstalling a curated set of tools for infrastructure work from one interface.

Current framing

  • Portable PowerShell and WPF desktop application
  • Handles install, update, and uninstall paths from the same checklist model
  • Includes offline USB kit building and import or export of custom selections
